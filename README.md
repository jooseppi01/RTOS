# RTOS

Kursilla käytetään Zephyrin RTOS käyttöjärjestelmää sulautetun laitteen ohjelmointiin, ohjelmointikielenä C++. <br>
Kehitysalustana käytämme nordic Semiconductorin nRF5340 Audio DK tai nRF5340 DK. Kurssin laitteet vaatii nRF Connect SDK:n asennuksen. <br>
Visual Studio Codea käytetään ohjelmointiympäristönä, nRF Connect-laajennukset löytyy VS codelle valmiina. <br>
Testiautomaatioon käytämme GoogleTest ja Robot Framework.<br>
